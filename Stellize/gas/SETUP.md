# お問い合わせフォーム受信のセットアップ手順（10分）

フォーム送信 → ①スプレッドシートに記録 ②rei.stella1127@gmail.com に通知
③相手に自動返信、が動くようになります。

Googleアカウント（rei.stella1127@gmail.com）での操作が必要なため、
以下はご本人にお願いする作業です。

---

## 手順

### 1. スプレッドシートを作る
1. https://sheets.new を開く（rei.stella1127@gmail.com でログインした状態で）
2. ファイル名を「Stellize お問い合わせ管理」などに変更

### 2. スクリプトを貼り付ける
1. メニューの「拡張機能」→「Apps Script」を開く
2. 最初から入っている `function myFunction() {...}` を全部消す
3. `contact-form.gs`（このフォルダにあるファイル）の中身を全部コピーして貼り付け
4. 上部の💾（保存）を押す。プロジェクト名は「Stellizeフォーム」などでOK

### 3. デプロイする（公開URLを作る）
1. 右上の「デプロイ」→「新しいデプロイ」
2. 歯車アイコン →「ウェブアプリ」を選択
3. 設定を以下にする：
   - 説明：任意（例「v1」）
   - 次のユーザーとして実行：**自分**
   - アクセスできるユーザー：**全員**  ← ここ重要！
4. 「デプロイ」を押す
5. 「アクセスを承認」→ 自分のアカウントを選択
   → 「このアプリは Google で確認されていません」と出たら
   「詳細」→「（プロジェクト名）に移動」→「許可」
   （自作スクリプトなので出る警告です。問題ありません）
6. 表示された **ウェブアプリのURL**（https://script.google.com/macros/s/～/exec）をコピー

### 4. URLをサイトに設定する
コピーしたURLを私（Claude）に渡すか、自分で設定する場合は
`contact.html` 内の以下の行に貼り付け：

```js
var FORM_ENDPOINT = "";  // ← ここに貼る
```

### 5. テスト送信
サイトのお問い合わせフォームから試しに送信して、
- スプレッドシートに行が増える
- rei.stella1127@gmail.com に通知が届く
- 入力したメールアドレスに自動返信が届く
の3つを確認してください。

---

## よくあるハマりどころ

- **「アクセスできるユーザー」を「全員」にし忘れる** → フォームから送れません
- **スクリプトを修正したとき** → 「デプロイ」→「デプロイを管理」→ ✏️ →
  バージョン「新バージョン」→「デプロイ」で更新（URLは変わりません）
- **Gmailの送信上限** → 無料アカウントは1日100通まで（自動返信+通知で1件2通消費。
  1日50件以上問い合わせが来るようになったら嬉しい悲鳴なので、その時に相談してください）
- **迷惑メール対策** → 自動返信がお客様の迷惑メールフォルダに入ることがあります。
  自動返信文にその旨の注意書きを入れる/独自ドメインメールに移行する等で改善できます

## セキュリティ・プライバシー

- スプレッドシートには個人情報が溜まります。**共有設定は「自分のみ」のまま**にしてください
- 公開前にプライバシーポリシー（privacy.html）の文面を必ず整備してください
