## 🎯 サイトコンセプト

Stellize株式会社の公式Webサイトは、**高級感と温かみを両立**させた信頼構築型コーポレートサイトです。

### デザインの特徴
- **一番星テーマ**: 星モチーフを全体に配置し、開花・成長を表現
- **手書き風水彩イラスト**: 多様な年代・性別・障害特性を自然に表現
- **70%ビジュアル + 30%テキスト**: イラスト中心の構成でメッセージを視覚的に伝達
- **3段構成**: タイトル + ビジュアル + 簡潔説明

### カラーパレット
- **背景**: 米色 `#FAF7F2`, `#F5F1EB`
- **メインカラー**: 深緑 `#5C7C6F`, 淡紫 `#AE9FC4`, 淡ピンク `#E8BFC8`
- **アクセント**: 星の金色 `#D4AF37`
- **文字色**: `#3A3A3A` (プライマリ), `#6B6B6B` (セカンダリ)

### フォント
- **見出し**: Noto Serif JP (上品な明朝体)
- **本文**: Noto Sans JP (読みやすいゴシック体)

## 🎨 デザインシステム

### コンポーネント
- **visual-card**: ビジュアル重視のカード（ホバー時に浮き上がり）
- **section-bg**: セクション背景（白 or クリーム色）
- **star-divider**: 星モチーフの区切り線
- **btn-primary / btn-secondary**: 統一されたボタンデザイン

### アニメーション
- **twinkle**: 星のキラキラアニメーション（3〜4秒サイクル）
- **float**: ゆっくりとした浮遊アニメーション（5〜6秒サイクル）
- **fade-in**: スクロール時のフェードイン効果

### レスポンシブ対応
- **1024px以下**: タブレット対応（2カラム → 1カラム）
- **768px以下**: スマホ対応（フォントサイズ調整、ナビゲーション最適化）
- **480px以下**: 小型スマホ対応（余白最小化、タッチ操作最適化）

**特徴**:

- 一貫した訴求力を実現

## 📝 コンテンツ方針

### トーン＆マナー
- **煽らない**: 営業色を抑え、信頼構築を重視
- **説教くさくない**: 上から目線ではなく、寄り添う姿勢
- **やさしく静か**: 温かみのある表現
- **前向き・主体的**: 利用者の主体性を尊重
- **「自己効力感」は使わない**: 専門用語を避け、わかりやすい言葉で

### 文章構成
- **50%削減**: 従来の長文を簡潔に
- **3段構成**: タイトル + ビジュアル + 簡潔説明
- **視覚優先**: イラスト70% + テキスト30%
